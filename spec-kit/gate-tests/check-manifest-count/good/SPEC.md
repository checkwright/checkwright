# Widget — a manifest whose counts all cite or are exempt

The widget's gates live in `gates.list`; the doc names that owner rather than
pinning a total, so nothing here restates a governed collection's size.

Thresholds and rates are rules, not totals: a suite needs at least three gates,
runs up to twelve checks, and ≥ 2 kits must vendor it; each stage fans out to
five gates per kit.

Partitions classify, they do not count: all but two stages are idempotent.

A cardinal named in `six gates` inline code is a meta-reference, and a fenced
example is skipped entirely:

```
This kit ships six gates and seven checks.
```

Singleton idioms use the word "one", outside the cardinal grammar: one owner
per fact, one iteration per kit.

The count of contracts is not a governed collection here, so honoring the four
contracts reads clean.
