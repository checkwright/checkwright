# Checkwright

This crate name is reserved for **Checkwright** — a coding-agent-assisted
delivery methodology shipped as installable kits: a self-testing gate SDK for
prose/spec/config surfaces, an evidence-stamped iteration lifecycle designed
for stateless agent sessions, and token-economics-aware context management.

Real releases will start at `0.1.0`. Until then, follow
<https://github.com/checkwright/checkwright>.
