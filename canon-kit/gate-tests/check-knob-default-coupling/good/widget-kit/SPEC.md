# widget-kit — SPEC

Config knobs and their defaults (the owning SPEC states each literal):

- `WIDGET_KIT_QUEUE_FILE` — default `TASK-QUEUE.md`, agreeing with every source site.
- `WIDGET_KIT_RUN_CAP` — positive integer, default 3.
- `WIDGET_KIT_MODE` — `exactly-one` (the default) or `at-most-one`.
